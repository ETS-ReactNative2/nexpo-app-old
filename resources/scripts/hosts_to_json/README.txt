		~~~ Host parser ~~~

	1. Install required package pandas with
	'pip3 install pandas'

	2. Put script in the same folder as hosts excel file

	3. Rename hosts excel file to 'hosts.xlsx'

	4. Run script with 'python3 host_script.py'

	5. You can then prettify the JSON file online with
	an editor of your choice by running
	
	'cat hosts.json | xclip --selection clipboard'

	and then pasting into JSON prettifier.
